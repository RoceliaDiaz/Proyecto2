Página web

¿Qué es la página web?

Una página web es un documento digital y accesible desde cualquier navegador con acceso a internet, y que puede incluir audio, video, textos, imágenes y sus diferentes combinaciones. Las páginas Web se encuentran programadas en un formato HTML o XHTML, y se caracterizan por su relación entre unas y otras a través de hipervínculos: enlaces hacia contenidos diversos que permiten una lectura compleja, simultánea y diversa, muy distinta a la que podemos hallar en los libros y revistas.

Tipos de páginas web

Páginas Web estáticas.Operan mediante la descarga de un fichero programado en código HTML, en el que están todas las instrucciones para que el navegador reconstruya la página Web, accediendo a las ubicaciones de sus elementos y siguiendo un orden preconcebido, rígido, que no permite la interacción con el usuario. Este tipo de páginas son meramente informativas, documentales, no interactivas. En otras palabras se trata de páginas en las que no se puede cambiar el contenido.

Páginas Web dinámicas. En este caso los usuarios pueden crear y colaborar en el contenido, a diferencia de las anteriores, las páginas Web dinámicas se generan en el momento mismo del acceso del usuario, empleando para ello algún lenguaje interpretado (como el PHP), lo cual le permite recibir solicitudes del usuario, procesarlas en bases de datos y ofrecer una respuesta acorde a sus requerimientos.


¿Cuáles son las páginas web más visitadas?

Esto gracias a la expansión de Internet, hoy en día, las personas pasan cada vez más tiempo navegando por distintas páginas web y buscando contenidos. Dentro de las páginas más visitadas podemos destacar las siguientes:

En México
Siendo el país con mayor número de hispanohablantes, los motores de búsqueda, las plataformas de video y ciertas redes sociales como Facebook lideran las visitas en México, junto también con las páginas web destinadas a ecommerce. Detrás de estos sitios siguen las páginas web de servicios de streaming.
En Chile
En Chile, por el contrario, las redes sociales adquieren un rol más preponderante, apareciendo entre las principales visitas de los usuarios de Internet junto con los motores de búsqueda. Luego, continúan los sitios web de las dependencias estatales, las páginas web de ecommerce y sitios institucionales de empresas nacionales.
En Colombia
Posteriormente a los tradicionales motores de búsqueda y plataformas de video, las páginas web más visitadas en Colombia se corresponden con la presentación de contenidos - particularmente deportivos - y ecommerce.
